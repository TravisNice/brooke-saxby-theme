A theme custom designed for Brooke Saxby and her Red Faces work.
 
== Description ==

This theme was custom made to have the principal features:
*	Pink colour theme
*	Mobile first and responsive
*	Vertical Side Bar for Navigation and Widgets that hides when the screen is small

= Docs & Support =

[Everyday Publishing](https://www.everydaypublishing.com.au/)

== Installation ==
There is no Automatic Installation at this time. The theme is not hosted in the WordPress SubVersion Repositories.

= Manual Installation =
Everyday Publishing will handle the installation of the theme from their GitHub repository until further notice that the theme is generalised into a version suitable for public use.
